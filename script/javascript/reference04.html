식별자란?

변수나 함수의 이름이 될 수 있는 프로그래머가 선언하는 단어.

[식별자 생성시 규칙]



    - 키워드 사용불가

    - 숫자로 시작 불가

    - 특수 문자는 _와 $만 허용

    - 공백 포함불가

[식별자 생성할 때의 관례]



    - 모든 언어가 사용 가능하나 알파벳 사용이 관례

    - 의미 있는 단어 사용

    - 생성자 함수의 이름은 대문자로 시작

    - 변수와 인스턴스, 함수, 메소드 이름은 소문자로 시작

    - 식별자가 여러 단어로 이뤄지면 각 단어의 첫 글자는 대문자

키워드란?

키워드는 제어문의 시작과 끝, 특정한 조작 목적 등으로 쓰입니다.이런 기능들이 정의되어 있기 때문에 식별자나 프로퍼티 이름으로 사용할 수 없습니다.

자바스크립트의 키워드 종류

키워드 설명
break loop문이나 switch문을 종료하기 위해 사용.
continue loop문을 빠져나가 가장상단으로 가는데 사용.
debugger 자바스크립트의 실행을 중지하고 (가능하다면)디버깅 function을 부른다.
do ... while do while반복문 실행
for for반복문 실행
function function을 선언한다.
return function에서 나온다.
var 변수를 선언한다.
